Simple Chatbot, Quiz, Recommendation, and Data Visualization System
============================================================

------------------------------------------------------------
Project Description
------------------------------------------------------------
This project is a Java Swing–based desktop application that integrates
multiple features into a single system:

- A simple rule-based chatbot for user interaction
- A quiz system with score-based recommendations
- Data visualization using bar and pie charts
- Java Swing GUI with AbsoluteLayout
- Event handling and basic analytics
- No database (data handled in-memory)

This application is designed as an academic and learning project to
demonstrate how different Java Swing components and external libraries
can be combined into a single desktop application.

System Requirements
------------------------------------------------------------

- NetBeans IDE (8.x / 12+)
- Operating System: Windows / macOS / Linux

------------------------------------------------------------
Libraries Used (Based on NetBeans Project)
------------------------------------------------------------
The following libraries are used in this project:

1. Absolute Layout
   - NetBeans Swing GUI Builder layout manager
   - Used for designing and positioning GUI components

2. JFreeChart 1.5.4
   - Required JAR: jfreechart-1.5.4.jar
   - Used for bar chart and pie chart data visualization

3. Java Development Kit (JDK)
   - Source level: JDK 8
   - NOTE:
     NetBeans may show "JDK 21 (Default)", but the project
     MUST be configured to run using JDK 8 for compatibility.

------------------------------------------------------------
How to Download JFreeChart
------------------------------------------------------------
1. Visit:
   https://www.jfree.org/jfreechart/

2. Download:
   jfreechart-1.5.4.zip

3. Extract the ZIP file

4. Locate:
   jfreechart-1.5.4.jar

------------------------------------------------------------
How to Configure JDK 8 in NetBeans
------------------------------------------------------------
IMPORTANT:
Even if NetBeans shows "JDK 21 (Default)", you must switch
the project to JDK 8.

Steps:
1. Go to Tools > Java Platforms
2. Click Add Platform
3. Locate your JDK 8 installation folder
4. Click Finish

Set JDK 8 for the project:
1. Right-click the project
2. Select Properties
3. Go to Libraries
4. Set Java Platform to:
   JDK 8
5. Go to Sources
6. Set Source/Binary Format to:
   JDK 8
7. Click OK

------------------------------------------------------------
How to Run the Project in NetBeans
------------------------------------------------------------

Step 1: Clone or Download the Project
------------------------------------
Option A: Clone using Git
- Team > Git > Clone
- Repository URL:
  https://github.com/kqye/Simple-Chatbot-and-Quiz-Program-with-recommendation-and-data-visualization.git

Option B: Download ZIP
- Download from GitHub
- Extract the project folder

Step 2: Open the Project
-----------------------
1. Open NetBeans
2. Click File > Open Project
3. Select the extracted project folder
4. Click Open Project

Step 3: Add JFreeChart Library
------------------------------
1. Right-click the project
2. Select Properties
3. Go to Libraries
4. Click Add JAR/Folder
5. Select:
   jfreechart-1.5.4.jar
6. Click OK

(Ensure it appears under Libraries in the project tree.)

Step 4: Run the Application
---------------------------
1. Locate the main JFrame file (contains main method)
2. Right-click the file
3. Select Run File
OR
4. Click the Run ▶ button in NetBeans
